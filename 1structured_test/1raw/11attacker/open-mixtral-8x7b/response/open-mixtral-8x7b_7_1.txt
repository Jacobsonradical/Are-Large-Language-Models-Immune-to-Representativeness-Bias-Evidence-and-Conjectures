- P(B) as the probability that a random person in the city is black (which we know is 15%)

P(A|B) = P(B|A) \* P(A) / P(B)

We are given that P(B) = 15% = 0.15, P(B|A) = 80% = 0.8, and P(B|A') = 20% = 0.2. Note that P(A) + P(A') = 1, so P(A') = 1 - P(A).

P(A|B) = 0.8 \* P(A) / 0.15

Then, we can find P(A) as follows:

P(A) = P(A|B) \* 0.15 / 0.8

P(A'|B) = P(B|A') \* P(A') / P(B)

We already know P(B) and P(B|A'), and we want to find P(A'), so we can rearrange the above equation as:

P(A') = P(A'|B) \* P(B) / P(B|A')

Since we know that P(A) + P(A') = 1, we can then find P(A) as:

P(A) = 1 - P(A')

P(A) = 1 - (P(A'|B) \* P(B) / P(B|A'))
